🚀 FASTEST METHOD (2 Minutes):

1. Install Railway CLI:
   npm install -g @railway/cli

2. Run these commands in THIS folder:
   railway login
   railway init
   railway up

3. Add API keys in Railway dashboard:
   Go to your project → Variables → Add:
   - OPENAI_API_KEY = (your key)
   - ANTHROPIC_API_KEY = (your key)

4. DONE! Get your URL:
   railway open

Your app is now LIVE at: https://your-app.up.railway.app

================================================================================

📋 ALTERNATIVE METHOD (Using GitHub):

1. Run: DEPLOY.bat
   (This prepares everything automatically)

2. Create GitHub repo and push:
   git remote add origin https://github.com/yourusername/captionfuck.git
   git branch -M main
   git push -u origin main

3. On Railway website:
   - New Project → Deploy from GitHub
   - Select your repository
   - Add environment variables (API keys)
   - Deploy!
